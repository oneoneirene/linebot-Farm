# line休閒農場機器人

1. 機器人id : @600sobud
2. 資料來源 : https://data.coa.gov.tw/Service/OpenData/ODwsv/ODwsvTravelStay.aspx
3. 使用說明 : 
在對話框中輸入「查詢 "農場名稱"」
例如：
查詢 春園休閒農場
查詢  丞翔園農莊民宿
即可查詢該農場或民宿的照片、地址、介紹、官方網址
